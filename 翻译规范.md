```

## 4.1 傅里叶级数
```

## 引用名词定义Glossary
[术语表](https://chrisniael.gitbooks.io/gitbook-documentation/content/format/glossary.html)

允许你指定术语并且在术语表中显示它们各自的定义。基于这些术语，GitBook会自动建立索引并高亮这些在文中的术语。
`GLOSSORY.md` 的格式非常简单：
```
# 术语
这个术语的定义

# 另外一个术语
它的定义可以包含粗体和其他所有类型的内嵌式标记...
```
